<h3>课程介绍：</h3>
<p>适合人群：</p>
<p>具备C++基础,窗口基础</p>
<p>课程目标：</p>
<p>学习OpenGL ES2.0可编程管线，以及OpenGLES2.0的特性，带领初学者入门。</p>
<p>课程简介：</p>
<p>零基础学习OpenGL ES2.0的课程，从无到有，从最进本的函数讲起，每一课时都附带一个例子程序。深入浅出的讲解可编程管线技术，令人费解的文理，以及混合技术，各种优化技术：顶点缓冲区，索引缓冲区，帧缓冲区，介绍精灵的使用，并使用shader制作粒子特效。</p>
